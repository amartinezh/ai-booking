# ============================================================
# chatbot-patterns.txt
# Palabras y frases que el chatbot reconoce sin llamar a Gemini.
#
# Formato:
#   - Líneas que empiezan con '#' son comentarios (ignoradas).
#   - Líneas en blanco son ignoradas.
#   - [greetings] → coincidencia EXACTA. Muestran bienvenida directa (sin "Sin problema").
#   - [farewell]  → coincidencia EXACTA (^...$). Solo en IDLE. Cierre cordial (agradecimiento).
#   - [goodbye]   → coincidencia EXACTA (^...$). EN CUALQUIER ESTADO y también por VOZ
#                   (se evalúa sobre la transcripción). El paciente quiere TERMINAR la
#                   conversación / colgar / "no agendar". Cierra con despedida cordial y
#                   resetea la sesión a IDLE. Distinto de [escape]: NO reabre el flujo.
#   - [escape]    → coincidencia EXACTA (^...$). REINICIO: "Sin problema. Empezamos de cero."
#                   (volver al menú / empezar otra vez), NO un cierre.
#   - [cancel]    → coincidencia de INICIO (^...). Inician flujo de cancelación DE UNA CITA.
#
# Diferencia clave (léase antes de mover palabras entre secciones):
#   [goodbye] = "me voy / chao / no quiero seguir"  → cierra la conversación.
#   [escape]  = "reinicia / volvamos a empezar / menú" → reabre el flujo desde cero.
#   [cancel]  = "cancelar mi cita"                    → anula una cita ya agendada.
#
# Nota: Las palabras de [greetings] también deben estar en [escape] para que
# el sistema las detecte como escape y evite llamar a Gemini.
#
# Para agregar una nueva palabra, añada una línea en la sección correcta
# y reinicie el servicio (o llame a reloadPatterns()).
# ============================================================
